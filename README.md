# Trader Behavior vs Market Sentiment

This project analyzes how Bitcoin market sentiment (Fear/Greed)
influences trader behavior and profitability on Hyperliquid.

Key Findings:
- Highest profitability during Greed markets
- Highest trade frequency during Fear markets
- Larger position sizes during extreme sentiment conditions
